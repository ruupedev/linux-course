# OSS, what and why
OSS short for Open Source Software is a term used to describe software that anyone can view, copy or modify. In plain terms, the source code for the software is accessible for anyone to use. Even thought open source can be lumped together, the licenses between different project can vary. Studies show that 97% of surveyed commercial codebases contain open source code. Open source projects can be a great way to level up your coding skills and increase your demand as a developer.

## Different kinds of open source
While all open source projects usually share common traits like free distribution, an open and editable source code and a licenses not specific to a product, not all open source is still the same. The license for these projects can differ from each other from public domain, that is basically a free for all license. To copyleft that forces you to publish your own projects also as copyleft if any code from such projects is used. In between these extremes is permissive and Lesser General Public Licenses (LGPL) that require only to republish the modified version of the "library" itself.

## Benefits of open source
As stated earlier in this summary, one benefit of open source is that it allows people to join the project and learn from them while contributing to the project. Also a community of likeminded people behind different open source projects can be beneficiary. On top of these there is the actual benefit of utilizing the code in your own projects cutting down costs and developement time.

## Free or Open?
A common misconception is that free software = open source software. While both are free to use there is one key difference. Free software has a copyright owener who has choosen to give the software away for free while the source code is still closed.

## Final words
All in all the whole developer community can thank Richard Stallman for pioneering with public licenses. Open source is a communal and non discriminating way to further develop the digital world we live in. Offering great learning opportunities and technologies absolutely free of charge. Some noteworthy open source software that have changed the world are Linux, WordPress and Android.
